### Commands :

#### Display help for skyflow CLI

```
Skyflow -h
```

```
Skyflow --help
```

#### Display version of current skyflow CLI

```
Skyflow -v
```

```
Skyflow --version
```

#### Generate skyflow configuration file

```
Skyflow init
```

#### Install module

```
Skyflow install
```

##### Optionnal -> List modules can be installed

```
Skyflow install --list
```
